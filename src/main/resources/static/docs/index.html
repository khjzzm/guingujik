1. 소개

여보야 점심 API는 사용자들이 점심 메뉴를 선택하고, 주문을 할 수 있게 도와주는 RESTful API입니다.

1.1. 목적

이 문서는 API의 사용법을 설명하고, 개발자들이 쉽게 여보야 점심 API를 통해 애플리케이션을 개발할 수 있도록 돕기 위해 작성되었습니다.

2. 시작하기

여보야 점심 API를 사용하기 위한 기본적인 가이드를 제공합니다.
